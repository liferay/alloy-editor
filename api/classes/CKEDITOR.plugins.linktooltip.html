    CKEDITOR.plugins.linktooltip Class

    Defined in: src/ui/yui/src/plugins/linktooltip.js:327

    CKEDITOR plugin which allows displaying tooltip over links in the editor. Internally it uses LinkTooltip. You may specify all properties which LinkTooltip accepts via editor.config.linktooltip property.
