    ButtonLeft Class

    Defined in: src/ui/yui/src/buttons/button-left.js:6

    The ButtonLeft class provides functionality for changing the alignment of an image.

    )
    protected

    Defined in src/ui/yui/src/buttons/button-left.js:35

    Handles the click event. If the current status of button is "pressed", applies "float: left" style to the image and removes "float" style otherwise.

    Parameters:

    • event EventFacade

      Event that triggered when user clicked on the button.

    updateUI

    ()

    Defined in src/ui/yui/src/buttons/button-left.js:12

    Updates "pressed" attribute of the button. If the currently selected element is an image and this image is explicitly left aligned, "pressed" attribute will be set to true, otherwise - to false.
