Text Alignment - Left

For the vintage writers out there, this button gives you left text alignment... as it should be.

Copy 
var editor = AlloyEditor.editable('myContentEditable', {
    toolbars: {
        styles: {
            selections: [
                {
                    name: 'text',
                    buttons: ['paragraphLeft'],
                    test: AlloyEditor.SelectionTest.text
                }
            ]
        }
    }
});
